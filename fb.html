Overview

This is a video game written in C using the Allegro5 framework. This is a pretty fun game, and I learned a lot making it.

How to run: Ensure Allegro5 is installed. Fork from git. Type: ./runsh, and ./a.out. This is untested.

Objective: Hit the ball, and avoid the fish. The fish spawn in random locations each hit, and you strike out if you hit any of them. In a sense, it is a simplified pinball game. You gain more points depending on how hard you hit the ball.
